# عالم البلاستك

موقع متجر إلكتروني لبيع المنتجات البلاستيكية والورقية

## المتطلبات
- Python 3.11.7
- Flask
- SQLAlchemy
- وباقي المكتبات في requirements.txt

## التثبيت
1. نسخ المستودع
```bash
git clone <رابط-المستودع>
```

2. تثبيت المتطلبات
```bash
pip install -r requirements.txt
```

3. إعداد قاعدة البيانات
```bash
python init_db.py
```

4. تشغيل التطبيق
```bash
python app.py
```
